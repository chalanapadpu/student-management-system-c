# 🎓 Student Management System (Python)

## 📌 Description
A simple student management system built using Python.  
This project allows users to store, view, and search student records using a JSON file.

## 🚀 Features
- Add student details (ID, Name, Marks)
- Display all students
- Search student by ID
- Data stored permanently using JSON file

## 🛠️ Technologies Used
- Python
- JSON (for file handling)

## ▶️ How to Run
1. Install Python on your system
2. Download or clone this repository
3. Open terminal in project folder
4. Run the program:python main.py
